# wikilaps

A web application for recording votes for Formula 1 race sessions, inspired
by [wikihoops.com](https://wikihoops.com). Using Rust + Axum + PostgreSQL +
sqlx for the backend and SvelteKit, Tailwind and Skeleton (component library)
for the frontend.

## AI usage note

I've used Claude for some code in this repository (mostly frontend and boring
stuff like CI) but the majority of the code (and this entire README) is hand-written.
